The Main Window

The EMS main window controls the currently active database, and cannot be closed. It allows the user to search for employees matching specific criteria, and open individual records for further examination or editing.

Auto-save

The employee management system automatically saves every change you make. You can see when the database was last saved in the application menu bar.

Employee Table

The employee table displays all of the employees in the database or the employees returned by a search.
You can view more detailed information about an employee by double-clicking on them.

You can change which columns are displayed by going to View > Columns and checking the columns you would like to view.

Searching

To search the employee database, click the Search... button. To clear your search, use the Show all button.

If you would like to view a single employee with a known ID, you can use the Find by ID... command, which automatically opens the employee's details window.
