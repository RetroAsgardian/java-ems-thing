Searching

The EMS search function is extremely versatile, allowing users to effortlessly create precise search queries for almost any criteria imaginable.

Search conditions are split into three parts: the field to check, the type of comparison to perform, and the value to compare against. The field and comparison type are both selectable from a drop-down menu, and the whole condition forms a statement in plain English, such as "First name starts with John".

When the search dialog opens, it contains only one search condition. The + button below the list adds another condition, and the - button removes the last condition. Search queries must always contain at least one condition.

By default, searches will return employees that match all of the given conditions. To instead find employees that match any condition, use the drop-down menu above the list.

To clear a search and display all employees in the database, select the Show all option in the main window.
